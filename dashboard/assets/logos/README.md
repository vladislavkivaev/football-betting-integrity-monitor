# League logos (optional)

By default the landing page shows a **country flag** for each league.

If you'd like to use real league badges instead, drop image files here named
by league code (lowercase), any of `.png` / `.svg` / `.webp` / `.jpg`:

```
d1.png   -> Bundesliga
e0.png   -> Premier League / EPL
t1.png   -> Süper Lig / Turkey
g1.png   -> Super League Greece
```

The app picks them up automatically — no code changes needed. Note that
official league logos are trademarked; only add artwork you have the rights
to use in this context.
