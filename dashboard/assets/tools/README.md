# Pipeline tool logos (optional)

By default the pipeline row shows generic icons standing in for each tool
(🐍 Python, 🐘 PostgreSQL, a "dbt" text badge, 🎈 Streamlit).

To use real logos instead, drop image files here named by tool key
(lowercase), any of `.png` / `.svg` / `.webp` / `.jpg`:

python.png       -> Python
postgresql.png   -> PostgreSQL
dbt.png          -> dbt
streamlit.png    -> Streamlit

The app picks them up automatically — no code changes needed, just refresh
the page. Logos are shown on a small white plate so transparent-background
artwork reads cleanly on the dark cards. Brand marks/logos may be trademarked
— only add files you have the rights to use in this context.
